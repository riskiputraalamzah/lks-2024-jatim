Instructions for Creating DOC Files

Welcome! This guide will show you how to run the PHP application that allows you to create DOC files from a title and content entered in a form. Follow these steps to get started:

Step 1: Ensure You Have PHP Installed

To run this application, you need to have PHP installed on your computer. If you don't have PHP installed, you can download it from the official website: PHP Downloads.

Step 2: Navigate to the Project Directory

Open your terminal (Command Prompt on Windows, Terminal on Mac/Linux), and navigate to the folder where the project files are located.

For example, if your project is in the "E2" folder, run the following command: 

cd path/to/E2

Step 3: Start the PHP Built-in Server

In the terminal, run the following command to start the PHP built-in server: 

php -S localhost:8000

This will start a local web server on http://localhost:8000.

Step 4: Open the Application in Your Browser

Open your web browser (Chrome, Firefox, etc.) and navigate to http://localhost:8000.

You should now see the form where you can enter a title and content for the DOC file.

Step 5: Create the DOC File

Fill in the form with a title and content, then click the "Create" button. The application will generate a DOC file based on the title and content you provided, and the file will be available for download.

Step 6: View the DOC File

Once the file is downloaded, you can open it in any text editor or Microsoft Word to view the content.

Additional Notes

  • If you encounter any issues, make sure your PHP installation is correctly configured.
  • You can stop the PHP server by pressing Ctrl + C in the terminal.
  • Ensure all the PHP files (such as index.php, create.php, and test_php_doc.php) are in the same directory.

Happy Coding!
